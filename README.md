# Bar La Esquina

Esta es una página estática de una sola página (single‑page) diseñada para **Bar La Esquina**, un bar de cócteles de autor.  

## Características

- **Diseño autocontenido**: todo el CSS está dentro de `<style>` y el JavaScript en `<script>`.
- **Tipografía**: *Bebas Neue* (display) y *Montserrat* (texto) cargadas desde Google Fonts.
- **Hero visual**: ilustración SVG de un vaso de cóctel con efecto neón.
- **Responsive**: se adapta a móviles, con menú hamburguesa.
- **Accesibilidad**: foco visible, `aria` attributes y respeta `prefers-reduced-motion`.
- **Sin frameworks ni build tools**.

## Estructura de archivos

- `index.html` – archivo principal con todo el contenido.
- `README.md` – este documento.

## Cómo usar

1. Copia los archivos a tu servidor o a una carpeta local.
2. Abre `index.html` en cualquier navegador moderno.
3. Modifica los textos y colores según necesites.
